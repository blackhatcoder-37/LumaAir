🎯 WHAT IS LUMAAIR?
Next-generation air quality forecasting platform that combines:
• 🤖 Machine Learning predictions
• 🌤️ Real-time weather data  
• 🏥 Health-focused solutions
• 📊 Interactive visualizations

🌍 Focus: Delhi, India air quality monitoring

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ KEY FEATURES:
🏠 Home Dashboard - ML forecasting with charts
🌤️ Weather Module - Temperature & weather data
🤖 AI Assistant - Google Gemini health chatbot  
🛍️ Product Recommendations - Air purifiers & masks
📊 Pretrained Data - Historical predictions (2019-2024)
📈 Live Monitoring - Real-time air quality tracking
⚙️ Settings - Multi-language, themes, preferences

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 TECHNICAL STACK:
• Python 3.8+ with Streamlit
• Google Gemini AI integration
• LightGBM machine learning models
• RapidAPI weather services
• Multi-language support (English/Hindi/Spanish/French)
• Responsive glass-effect UI design
